# Master Design Suite

This skill is the creative engine of Synapse. It intelligently orchestrates the design process, from wireframing and UX research to high-fidelity UI generation and frontend coding.

## 🚦 INTELLIGENCE DISPATCHER

The system uses `scripts/dispatcher.py` to automatically detect your design needs:

- **LAYER 1: UX STRATEGY & PLANNING**
    - Detects: "wireframe", "sitemap", "user flow", "UX research", "accessibility".
- **LAYER 2: VISUAL & CREATIVE DESIGN**
    - Detects: "generate image", "icon", "logo", "color palette", "typography", "AI artist".
- **LAYER 3: FRONTEND ARCHITECTURE**
    - Detects: "React component", "Tailwind", "CSS", "Frontend code", "layout".

## 🛠️ DESIGN WORKFLOWS

### 1. From Concept to Visuals
Using the **AI Artist** and **UI/UX Pro Max** layers to generate high-fidelity mockups, moodboards, and branding assets from a simple text prompt.

### 2. Design to Code (Google Stitch)
Leveraging **Google Stitch** and **Frontend Design** layers to transform visual concepts directly into clean, responsive React/Next.js code following the best engineering practices.
